# BookBuddy

BookBuddy е минималистично уеб приложение за **организиране на лична библиотека** и **следене на прогреса при четене**. Работи изцяло във браузъра с **vanilla HTML/CSS/JavaScript** и **localStorage** (без backend, без библиотеки).

## Основни функции
- ✅ Добавяне на книга (заглавие, автор, общо страници, текуща страница, бележки)
- ✅ Редакция на книга
- ✅ Изтриване на книга
- ✅ Автоматичен прогрес бар и процент
- ✅ Статистика: брой книги, прочетени страници, среден прогрес
- ✅ Логване на четене (“Логвай четене”) + “Последна активност” (activity feed)
- ✅ Данните се пазят в `localStorage` и остават след refresh

## Технологии
- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage (persist)
